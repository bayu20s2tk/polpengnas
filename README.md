# Laravel CMS Starter
A Laravel Website with Admin access Starter project with AdminLTE theme and basic features.

## Features
 - Admin LTE theme
 - Authentication
 - User Roles
 - Admin Navigation
 - Log and View Activities (website actions and admin resource changes)
 - Accounts
 - App Settings
 - Banners
 - Page Builder
 - Resources (Documents, Photos, Videos)
 - News
 - Shop
 - FAQ
 - Website: Contact Us
